# 口令只在刮开完成、正式打卡时弹出

旧逻辑"第一次触碰徽章就弹口令"，导致手机双指缩放被当误触立刻弹窗，且刮开画布 `touch-action:none` 屏蔽了双指缩放。决定（2026-06-30）：只在某个徽章刮开完成、要正式点亮时才要口令；取消则箔层复原。口令一个会话只问一次（私钥解锁后驻留内存）。画布 `touch-action` 改为 `pinch-zoom`，检测到第二根手指即停止刮开、放行浏览器缩放。

## Consequences

刮开过程不被打断、不误触。注：决策当时完成阈值为"刮够一半"，后已改为刮满整块（TH=1，容差 1 格），口令时机不变——始终是"完成时"。
