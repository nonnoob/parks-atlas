# 州视图：保留州轮廓底图，徽章缩小并约束在州界内

徽章是固定大小、非真实地理比例的，与真实比例的州轮廓底图叠加时，小公园会"超出州界"并互相遮挡名字（如国王峡谷）。决定（2026-06-30）：保留州轮廓底图，按州的实际尺寸缩小徽章，并把位置约束在州轮廓内（超界则向州质心收拢），垂直方向为名字留高。

## Considered Options

- 去掉州底图 —— 最干净但丢失地理感，被否。
- 底图变淡只做装饰 —— 被否。

## Consequences

公园密集的州（加州 9 座）徽章偏小、名字轻微靠近。可接受：礼物向，趣味 > 制图精度。后续演进：地图内详情卡取代了徽章旁名字，名字只在详情卡显示。
